SJF Version Note

This project includes both SJF versions. Non-Preemptive SJF selects the shortest arrived process only when the CPU becomes free and then runs it to completion. Preemptive SJF, also called SRTF, always selects the process with the shortest remaining time and may interrupt the current process when a shorter job arrives.

Run all three algorithms to compare average metrics.
